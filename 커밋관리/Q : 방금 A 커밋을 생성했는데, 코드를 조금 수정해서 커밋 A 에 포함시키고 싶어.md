**Q : 방금 A 커밋을 생성했는데, 코드를 조금 수정해서 커밋 A 에 포함시키고 싶어**  
A : Commit Window > Changes 트리에서 커밋에 포함시킬 파일 또는 코드 라인 선택 > Amend 체크박스 선택 > Amend Commit and Push > Force Push
